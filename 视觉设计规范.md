# 个人主页 — 视觉设计规范

基于 Solo Builder 定位的完整视觉设计系统，供设计 Agent 参考并生成高保真设计稿。

---

## 一、品牌定位与调性

| 维度 | 内容 |
|------|------|
| **角色** | Solo Builder — 独立建设者、写代码、写文章、做产品 |
| **受众** | fellow builders、潜在合作者、好奇的读者 |
| **调性** | 真实、松弛、技术人气质；现代、清爽、有呼吸感 |
| **禁忌** | 企业味儿、过度设计、视觉喧嚣 |

## 二、色彩系统（已确认）

### 2.1 亮色模式 Light Theme
```
背景：#FFFFFF (或 #FAFAFA 微暖白)
文本主色：#1F2937 (深灰)
品牌主色：#0066CC (蓝色) → 用于 CTA、链接、强调
品牌次级：#0052A3 (深蓝) → hover/active 状态
卡片背景：#F3F4F6 (浅灰)
边界线：#E5E7EB (极浅灰)
```

### 2.2 暗色模式 Dark Theme（已确认偏好：深灰 #1A1A1A）
```
背景：#1A1A1A (深灰)
卡片背景：#252525 (更深的灰)
文本主色：#E5E7EB (浅灰)
品牌主色：#4B9FFF (亮蓝) → 与深背景形成对比
品牌次级：#3A8FE6 (稍深蓝) → hover/active
边界线：#333333 (超浅灰)
```

### 2.3 颜色用途
- **#0066CC / #4B9FFF**：CTA 按钮、链接、强调文字、表单焦点、标签
- **绿色（可选）**：Success/Released 状态
- **黄色（可选）**：Warning/Development 状态  
- **灰色梯度**：Disabled、次要信息、边界

## 三、排版系统

### 3.1 字体
- **西文**：Inter（Google Fonts，所有权重 400/500/600/700）
- **中文**：Noto Sans SC（Google Fonts，权重 400/500/600/700）
- **代码**：Fira Code 或 JetBrains Mono（等宽）

### 3.2 字体大小与行高
| 层级 | 桌面 | 行高 | 用途 |
|------|------|------|------|
| **H1** | 48px | 1.2 | 页面标题、Hero 身份 |
| **H2** | 36px | 1.3 | 模块标题（Builds, Projects 等） |
| **H3** | 28px | 1.4 | 卡片标题、子标题 |
| **Body Large** | 18px | 1.6 | 副文本、描述 |
| **Body** | 16px | 1.6 | 正文、默认 |
| **Body Small** | 14px | 1.5 | 辅助信息、标签、日期 |
| **Caption** | 12px | 1.4 | 极小信息、代码行号 |

**移动端调整**：基础字号减小 2-4px，保持行高不变。

## 四、间距系统（8px 基数）

```
xs:  4px
sm:  8px
md:  16px
lg:  24px
xl:  32px
2xl: 48px
3xl: 64px
```

### 常见应用
- **卡片内边距**：md / lg（16-24px）
- **模块间距**：xl / 2xl（32-48px）
- **列表项间距**：md（16px）
- **按钮内边距**：sm md（8px 16px）

## 五、圆角与阴影

### 5.1 圆角半径
```
卡片、大按钮：8px
小按钮、输入框：6px
极细节：4px
```

### 5.2 阴影（轻量级）

| Medium | `0 4px 12px rgba(0,0,0,0.15)` |
| Heavy | `0 20px 25px rgba(0,0,0,0.15)` |

## 六、页面模块设计

### 6.1 Header（固定顶部导航）
```
┌─────────────────────────────────────────────┐
│ [Logo] 名字      Builds Projcets ... | 🌙   │  ← 固定高度 64px
└─────────────────────────────────────────────┘
```

- **高度**：64px（含上下内边距）
- **背景**：
  - 页面顶部：完全透明或白/黑色
  - 向下滚动：半透明毛玻璃（backdrop-filter: blur(10px) + rgba 背景）
- **导航链接**：
  - 文本：Body Small 或 14px
  - 当前 section 高亮：使用品牌主色或 underline
  - hover：颜色加深
- **Dark mode toggle**：
  - 右侧 icon（月亮/太阳），可点击切换
  - Hover：背景轻微变化

**毛玻璃效果**（向下滚动激活）：
```css
backdrop-filter: blur(10px);
background: rgba(255, 255, 255, 0.85); /* 亮色 */
/* 暗色: rgba(26, 26, 26, 0.85) */
```

### 6.3 Builds 模块（作品展示）
```
┌──────────────────────────────────────────┐
│  Builds | 我的作品                        │ ← H2 36px
├──────────────────────────────────────────┤
│                                          │
│  ┌────────────────────────────────────┐  │
│  │                                    │  │
│  │    [Logo/主视觉图] 800×400px      │  │ ← 大卡片
│  │                                    │  │
│  │    Project Name (英文)            │  │ ← H3
│  │    中文描述或英文 tagline         │  │
│  │    技术栈：React, Node, ...       │  │
│  │    [GitHub] [Demo]               │  │
│  │                                    │  │
│  └────────────────────────────────────┘  │
│                                          │
│  ┌──────────┐ ┌──────────┐             │
│  │          │ │          │             │
│  │ 小卡片    │ │ 小卡片    │             │ ← 2×2 网格
│  │ 300×200  │ │ 300×200  │             │   （桌面）
│  │          │ │          │             │
│  └──────────┘ └──────────┘             │
│  ┌──────────┐ ┌──────────┐             │
│  │ 小卡片    │ │ 小卡片    │             │
│  └──────────┘ └──────────┘             │
│                                          │
└──────────────────────────────────────────┘
```

**设计要点**：
- **大卡片**：
  - 尺寸：full-width（留边距）or 800×400px
  - 图片 top，下方 padding lg 放标题、描述、链接
  - 悬浮时：translateY(-4px) + shadow 增强（0.3s）
  - 圆角：8px
  
- **小卡片**：
  - 尺寸：每个 300×200px 或响应式等宽
  - 同样的悬浮动效
  - 图片占满卡片、cover fit，文字 overlay 或底部
  
- **响应式**：
  - 平板（1024px）：1 大卡 + 2×2 小卡变 1 列
  - 移动（<768px）：全部堆叠单列

**设计要点**：

## 八、交互与动画

### 8.1 卡片悬浮效果
```css
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
/* Hover 状态 */
transform: translateY(-4px);
box-shadow: 0 20px 25px rgba(0, 0, 0, 0.15);
```

### 8.2 导航平滑滚动
```css
scroll-behavior: smooth;
```

### 8.3 Dark Mode 切换
- 过渡时间：0.3s
- 使用 CSS 变量实现（不刷新页面）
- 可选：淡入淡出效果

### 8.4 按钮状态
```
Normal     → 品牌主色背景 + 白文字
Hover      → 更深的主色 or 阴影增强
Active     → 压低感（可选 transform）
Focus      → outline (2px 品牌主色，offset 2px)
Disabled   → 灰色背景 + 灰文字，opacity 0.5
```

### 8.5 链接悬浮
```
Normal     → 品牌主色 + 无下划线 or 细下划线
Hover      → 下划线加粗 or 颜色变深
Visited    → 可选：灰色淡化
```

## 九、响应式断点

| 断点 | 设备 | 调整 |
|------|------|------|
| ≥ 1200px | 桌面 | 多列布局，2-4 列卡片，目录 sidebar |
| 1024px - 1199px | 平板 | 2 列或单列混合，导航可变为汉堡菜单 |
| 768px - 1023px | 大手机 | 单列为主，汉堡菜单 |
| < 768px | 小手机 | 100% 单列堆叠，极简导航 |

**关键调整**：
- **导航**：桌面文字链接 → 平板/移动汉堡菜单
- **卡片**：2-4 列 → 2 列 → 1 列堆叠
- **表格**：行式 → 卡片行（mobile card list）
- **Sidebar**：右侧 sidebar → Drawer/浮窗

## 十、可访问性 (WCAG AA 标准)

- **色彩对比度**：≥ 4.5:1（正文 vs 背景）
- **焦点状态**：所有交互元素有明显 focus outline
- **图片**：所有图片含 alt 文本
- **链接**：链接文本清晰，不仅靠颜色区分
- **表单**：input 有 label，error message 清晰
- **动画**：支持 `prefers-reduced-motion` 媒体查询

---

## 十一、性能优化

- **字体加载**：Google Fonts，使用 font-display: swap
- **图片**：WebP + JPG fallback，使用 srcset 响应式
- **Syntax Highlighting**：轻量级库（highlight.js 或 Prism.js）
- **Lighthouse 目标**：≥ 90 分
